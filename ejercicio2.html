1) Lista de “misiones” del curso

Lista no ordenada personalizada (cuadrados):

  • Crear estructura básica de una web (index)
  • Aprender etiquetas principales (h1, p, a, strong, i)
  • Trabajar con listas
    • Listas no ordenadas (ul)
    • Listas ordenadas (ol)
    • Listas anidadas
  • Tablas y estructura
    • Filas (tr) y celdas (td)
    • Encabezados (th)
    • Uniones (colspan / rowspan)
  • Introducción a elementos de entrada (input)

Lista ordenada (romanos) con un “plan de trabajo”:

  1. Repasar teoría
  2. Hacer un ejemplo propio
  3. Subirlo a GitHub Pages
  4. Comprobar que los enlaces funcionan
